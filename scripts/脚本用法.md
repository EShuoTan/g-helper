# build.ps1

脚本会自动停止正在运行的GHelper进程，发布单个exe文件，并复制到指定目录。

```powershell
# 默认构建（Release配置，win-x64，输出到D:\app\g-helper）
.\scripts\build.ps1

# 自定义参数
.\scripts\build.ps1 -Configuration Debug -Runtime win-arm64 -OutputDir "C:\output"

# 清理后构建
.\scripts\build.ps1 -Clean
```
